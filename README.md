# Varatraza Tech — Agence Digitale IA 🇲🇬

Varatraza Tech est une agence digitale premium basée à Madagascar, spécialisée dans la transformation numérique des entreprises (hôtellerie, restauration, services).

## 🚀 Vision
Propulser le business local vers les standards internationaux grâce à l'Intelligence Artificielle et aux automatisations intelligentes.

## ✨ Services
- **Création de Sites Web** : Vitrines multilingues, e-commerce et portfolios hautes performances.
- **Chatbots IA** : Assistants intelligents entraînés sur vos données pour un support 24h/24.
- **Automatisation RPA** : Optimisation des processus métiers avec n8n.
- **Marketing IA** : Stratégies de contenu et outils de capture de leads automatisés.

## 🛠️ Stack Technique
- **Frontend** : React, Vite, TypeScript, Tailwind CSS, Framer Motion.
- **IA** : OpenAI GPT-4o API.
- **Icons** : Lucide React.
- **Fonts** : DM Sans.
